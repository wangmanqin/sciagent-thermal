# SciAgent 微通道散热器评测基准

> 共 10 道题，按难度递进，从传热基础到多目标优化设计。

## 第1题 | 矩形肋片效率 | easy

**问题：**
计算矩形肋片的效率。已知：

- 肋片材料：硅（导热系数 k = 148 W/mK）
- 肋片高度 H = 0.3mm
- 肋片厚度 t = 0.05mm
- 对流换热系数 h = 10000 W/m²K

用肋片效率公式 η = tanh(mH)/(mH) 计算，其中 m = √(2h/(k·t))。
画出肋片效率随高度 H（0.1~1.0mm）变化的曲线图。

**标准答案：**

- m = √(2×10000 / (148×0.00005)) = **1644 m⁻¹**
- mH = 1644 × 0.0003 = **0.493**
- η = tanh(0.493) / 0.493 = **0.921（即92.1%）**
- 效率随 H 增大而降低

**物理意义：** 微通道的侧壁可以看作肋片，肋片效率反映热量能传多远。效率越高，壁面散热越均匀。**核心的内容是在测试肋片到底做多长、做多厚、用什么材料，才最划算。**

## 第2题 | 圆管层流Nu数验证 | easy

**问题：**
验证圆管内充分发展层流（恒壁温）的努塞尔数 Nu = 3.66。条件：

- 管径 D = 0.5mm，管长 L = 10mm
- 流体为水（kf = 0.613 W/mK, μ = 8.9×10⁻⁴ Pa·s, ρ = 997 kg/m³）
- 流速 u = 0.5 m/s

计算 Re 确认为层流，计算 h，画 h 随流速变化曲线。

**标准答案：**

- **Re** = ρuD/μ = 997×0.5×0.0005 / 8.9×10⁻⁴ = **280（<2300，确认层流）**
- **Nu** = **3.66**（经典解析解）**圆管内层流、充分发展流动 → Nu = 常数 3.66**
- **h** = Nu·kf/D = 3.66×0.613/0.0005 = **4490 W/m²K**
- 层流时 h 与流速无关（Nu 是常数）

**物理意义：** Nu 数表征对流换热强度。层流充分发展时 Nu 为常数，这是传热学的基本结论。

**Re < 2300** → **层流（流动很规整、平稳）----**Re > 4000** →  **湍流** （混乱、漩涡多、换热强）**

## 第3题 | 矩形通道摩擦系数 | easy

**问题：**
计算矩形微通道的水力直径和摩擦系数。

- 通道宽 Wc = 0.1mm，高 Hc = 0.3mm
- 流体为水，流速 u = 1 m/s

用 Shah-London 关联式计算 Poiseuille 数 Po，画 Po 随宽高比 α 变化曲线。

**标准答案：**

- 水力直径 **Dh** = 2×Wc×Hc/(Wc+Hc) = **0.15mm**
- 宽高比 α = Wc/Hc = **0.333**
- Re = ρuDh/μ = **168**
- Po(α=0.333) = **68.38**（Shah-London公式）------Po = 96 × (1 - 1.3553α + 1.9467α² - 1.7012α³ + 0.9564α⁴ - 0.2537α⁵) 微通道矩形通道
- 摩擦系数 f = Po/Re = **0.407**
- α=1（正方形）时 Po=56.91

**物理意义：** 摩擦系数决定了流体在通道中的阻力大小，直接影响压降和泵功率。

## 第4题 | 微通道压降与泵功率 | medium

**问题：**
计算矩形微通道的压降和泵功率。

- 通道 0.1mm×0.3mm×10mm长
- 水，流速 u = 1 m/s

用 ΔP = 2f·ρu²L/Dh 计算压降，再算泵功率。画压降和泵功率随流速变化的双Y轴图。

**标准答案：**

- ΔP = 2×0.407×997×1²×0.01/0.00015 = **54096 Pa ≈ 54.1 kPa**
- 单通道流量 Q = u×Wc×Hc = **3×10⁻⁸ m³/s**
- 单通道泵功率 Wp = ΔP×Q = **1.62 mW**
- 层流中：压降∝流速，泵功率∝流速²

**物理意义：** 压降是微通道设计的关键约束——通道越窄压降越大，需要更多泵功率。

## 第5题 | 热阻网络模型（核心题）| medium

**问题：**
用一维热阻网络模型计算散热器总热阻。

- 硅基底 t=0.1mm，通道 0.1mm×0.3mm，壁厚 0.05mm
- 芯片 10mm×10mm
- h 由 Nu=4.0 计算

R_total = R_cond（导热热阻）+ R_conv（对流热阻）

**标准答案：**

- h = Nu·kf/Dh = 4.0×0.613/0.00015 = 16347 W/m²K   **kf是导热系数**
- 通道数 N = 10mm/0.15mm = **66个**
- 肋片效率 η = **0.886**
- 导热热阻 R_cond = 0.1mm/(148×10⁻⁴m²) = **0.00676 K/W**
- 对流热阻 R_conv = **0.147 K/W**
- **总热阻 R_total = 0.154 K/W**

**物理意义：** 这是 Tuckerman & Pease (1981) 微通道散热经典模型的简化版。总热阻决定了芯片能冷却到什么温度。对流热阻远大于导热热阻，说明**瓶颈在对流换热**。

## 第6题 | Nu随宽高比的参数研究 | medium

**问题：**
研究矩形通道宽高比 α 对 Nu 的影响。
用关联式 Nu = 8.235×(1 - 2.0421α + 3.0853α² - ...)，画 Nu 和 f·Re 随 α 变化的双Y轴图。

**标准答案：**

- α=1（正方形）：Nu = **3.608**
- α=0.333：Nu = **4.795**
- α=0.1（很扁的通道）：Nu = **6.785**
- α→0（平行平板）：Nu→**8.235**
- **通道越窄，Nu越大，换热越强**

**物理意义：** 窄通道虽然换热更好，但阻力也更大。这个参数研究帮助理解几何形状对性能的影响。

## 第7题 | 单目标优化（有约束）| medium

**问题：**
优化通道宽度 Wc，使总热阻最小，**约束泵功率 ≤ 200mW**。

- Wc 范围：0.02~0.3mm
- 其他参数固定

**标准答案：**

- 最优 Wc ≈ **0.065mm**
- 最小热阻 R ≈ **0.095 K/W**
- 对应泵功率 ≈ **196 mW**（刚好在约束边界上）
- Wc 越小 → 热阻越低，但泵功率急剧上升
- **最优解在约束边界上**（泵功率刚好用完）

**物理意义：** 这就是工程优化的典型场景——散热和泵功率不能同时最好，需要在约束下找平衡。

```
Wc 小 → 散热好（热阻低）  但 泵功率高（压降大）
Wc 大 → 泵功率低（压降小）但 散热差（热阻高）
```

## 第8题 | 整体性能计算 | easy

**问题：**
已知散热器参数（66通道，ΔP=54.1kPa，R=0.154K/W），计算：

- 总泵功率
- 芯片最高温度（功率100W）
- 水的温升

**标准答案：**

- 总泵功率 = 66×54100×3×10⁻⁸ = **107 mW**
- 芯片温度 T = 25 + 100×0.154 = **40.4°C**
- 水温升 ΔT = 100/(997×4179×66×3×10⁻⁸) = **12.1°C**
- 出口水温 = 25 + 12.1 = **37.1°C**

**物理意义：** 把前面所有计算串起来，得到散热器的最终性能指标——这就是设计报告里要写的数字。

## 第9题 | NSGA-II 双目标优化 | hard

**问题：**
用 NSGA-II 同时最小化**热阻**和**泵功率**。

- 设计变量：Wc∈[0.03,0.3]mm, Hc∈[0.1,0.8]mm, u∈[0.5,3.0]m/s
- 种群100，迭代200代
- 画 Pareto 前沿图

**标准答案：**

- Pareto前沿呈现 **trade-off 关系**：
  - 低热阻 ↔ 高泵功率（窄通道、高流速）
  - 低泵功率 ↔ 高热阻（宽通道、低流速）
- 前沿为光滑连续曲线

**物理意义：** 这是你研究的核心——没有一个方案能同时让热阻和泵功率都最小，Pareto 前沿展示了所有"不浪费"的设计方案，设计师根据实际需求在前沿上选点。

## 第10题 | 完整设计优化 | hard

**问题：**
给定约束条件的完整设计任务：

- 芯片 10mm×10mm，功率 150W
- **要求 T_max ≤ 80°C**
- 目标：**泵功率尽可能小**

找满足温度约束下泵功率最小的设计方案。

**标准答案：**

- 温度约束 → R_total ≤ (80-25)/150 = **0.367 K/W**
- 最优方案应在**约束边界附近**（R刚好≈0.367，泵功率最小）
- 需输出最优的 Wc、Hc、u 以及对应的所有性能指标

**物理意义：** 这是实际工程设计的完整流程——给定芯片参数和温度要求，设计出最省功率的散热器。这道题综合了前面所有知识。
